# Chapter 3: Variables for Numbers

This repository contains JavaScript assignments related to variables for numbers from the Saylani Mass IT Training program, Techno Kids Batch 7.

## Overview

This project covers the fundamentals of using variables to store and manipulate numeric values in JavaScript. It includes various exercises that demonstrate how to work with numbers, perform arithmetic operations, and understand numeric data types.

3. The JavaScript code in the `script.js` file will execute and display results
