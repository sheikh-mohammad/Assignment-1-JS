# Question 1 – Variables for Strings

This exercise practices declaring string variables and displaying them via the browser and/or console.

## Files
- `index.html`: Minimal HTML that includes `script.js`
- `script.js`: JavaScript for this question

## How to Run
- Open `index.html` in your browser
- Or use VS Code Live Server: right‑click `index.html` → Open with Live Server

## Tips
- Use `console.log(...)` to inspect variable values
- Try changing the string values and concatenating multiple strings

## Expected Learning
- Declare and initialize string variables
- Concatenate strings and display results
