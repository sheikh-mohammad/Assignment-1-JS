# Chapter 1: Alerts

This folder contains JavaScript assignments related to alerts in Chapter 1.

## Overview
A beginner-friendly collection of JavaScript exercises focused on implementing alert dialog boxes. Each question introduces fundamental concepts of displaying messages to users through JavaScript alerts, forming the foundation for user interaction in JavaScript programming.

## Contents
- **Question 1**: Displays a welcome alert message
- **Question 2**: Alert implementation
- **Question 3**: Alert implementation
- **Question 4**: Alert implementation
- **Question 5**: Alert implementation

Each question directory contains:
- `index.html`: HTML file to run the JavaScript code
- `script.js`: JavaScript file containing the alert implementation

## Purpose
This chapter focuses on learning how to use JavaScript alert boxes to display messages to users.

## Learning Objectives
- Understanding the basic syntax of alert() function
- Implementing simple message dialogs
- Creating user notifications with alert boxes
- Learning the difference between alert, prompt, and confirm functions
- Getting familiar with JavaScript debugging using alerts

## Alert Functions in JavaScript
- `alert()`: Displays a message with an OK button
- `prompt()`: Asks the user for input with a text field
- `confirm()`: Asks the user to confirm an action with OK/Cancel buttons
